deface|

Minimal Facebook chat for chrome and firefox

If you don't want to get distracted by Facebook, or throw away your privacy with the mobile messenger app, but still need to chat, try deface.


features

  • remove everything except the chat from facebook.
  • increased chat-box size
  • toggle deface off/on with the icon in the top-left.
  • 'quick peek' mode to prevent you from getting stuck in the stream
  • please login to Facebook to use deface.
  • expect bugs, this is a young project :)
